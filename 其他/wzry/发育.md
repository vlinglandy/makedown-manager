## 目标就是发育和推塔

```mindmap
- 打法
  - 攻速鞋
  - 前期利用手长的优势进行消耗
    - 3蹲中路，打野没来
      - 中路来了，对a想办法杀
    - 防打野，中路来了守着
    - 自己方打野来了
      - 稳住，后手赢
      - 不要把对方逼得太紧
```

## 消耗

```mindmap
- 消耗
  - 偷袭
    - 躲草之后进攻
    - 打兵顺便打人（要求同时打三个兵）
  - 光明正大
    - 看对方控制技能交了之后上
    - 对撸，存一发子弹，等1s
      - 打不过进草丛
      - 打过
```

## 对撸

```mindmap
- 流程
  - 看对方装备
    - 对攻速就上
  - 看对方打法
    - 看技能
    - 看清兵
  - 躲控制
    - 先存一发

```

> 先清线，然后给假视野
>
> 回来位移进草丛，存一发
>
> 看对方技能，走进，打一枪
>
> 躲控制，继续二技能，前位移，打一枪
>
> 走a，打不过就走
